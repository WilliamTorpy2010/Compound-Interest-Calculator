Compound Interest Calculator

This Python program calculates the future value of an investment using the principle of compound interest. 

The user inputs:
1. Initial investment amount (currency)
2. Annual interest rate (%)
3. Investment duration (years)

The program then calculates how the investment grows over time, demonstrating the effect of compounding returns and allowing users to understand how long-term investments can increase in value.

Features:
- User input 
- Compound interest calculations
- Clear output showing final investment value and interest earned

  Why this is significant: Compound interest is a key concept in finance and economics as it demonstrates how money grows over time through reinvested returns which compound. As interest rates change, individuals and businesses adjust their decisions about saving, investing, and borrowing. Higher interest rates can encourage people to save more due to higher returns on savings accounts, while also making borrowing more expensive. This can reduce consumption and business investment, which contributes to slower economic growth.
  Aim of the program: The aim of this program is to help users estimate the potential growth of their savings through compound interest. By changing variables such as the initial investment, interest rate, and investment period, users are able to use the data to make informed financial decisions  which could influence future returns.
